# AnotherOnlineMode 
###### A BungeeCord plugin of checking players' account.

---
### Command:
    /aomreload:
        description: Reload config.
        permission: anotheronlinemode.admin

    /zb: (default)
        description: Check your account.

### NO limits: `permission: "anotheronlinemode.bypass"`

### Version: `0.0.2`
